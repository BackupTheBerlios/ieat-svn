iEat  Version 1.0 – User Guide

 

               What is iEat?

 

iEat is a recipe storage, sharing and management application for the twenty first century. In an age where any and everything is done everything electronically why are we forced to store recipes on hand written sheets or in books which could be easily damaged or destroyed? iEat will give users a simple means to enter and store their recipes in a digital medium. Users will be able to share recipe’s with their friends with the click of a button and will be able to easily generate shopping lists for a set of selected recipes that they plan to cook. If users choose to photograph their creations then they can store the photos (and possibly in the future videos) right along with the recipe.

 

               Getting Started

 

iEat was created with the user in mind, so hopefully you will find most tasks intuitive and natural and should not have to reference this guide to perform most tasks. When you start the application you will be presented with the iEat recipe file which displays all your recipes along with a summary of each recipes details.

 

You have several options from this main screen.

 

The controls are broken down into 3 main sections:

·        The Main Toolbar – Allows access to editing/viewing/printing recipes and adding new recipes. This toolbar is located at the top of iEat directly above the recipes.

·        The Sort/Filter Pane – Allows you to filter/sort the recipes displayed in the main window. The control pane is located directly to the left of the recipes.

·        The Control Menu – Allows you to access most of the options on the main toolbar but in menu form. The menu's also allow access too the help menu in addition to allowing access to this guide.

·        The Search Field – Allows you to search for recipes using any of the entered details. The search field is located at the top of iEat to the left of the Main Toolbar.

 

               Entering/Editing Existing Recipe

 

In order to create a new recipe either use the “New Recipe” button or use the File->New Recipe menu option.

 

To edit an existing recipe simply select the desired recipe in the main window and then select either the “Edit Recipe” button or user the File->Edit Recipe menu option.

 

Weather your creating a new recipe or editing an existing one you will be shown the Recipe Edit/Creation window. This window allows you to enter in all the recipe information that can currently be stored/managed via iEat.

 

Recipe Information:

·        Recipe Name – Should be something descriptive to aid you in identifying the recipe in the future.

·        Tags – These are searchable tags that can be used to categorize recipes. For example, tags for my favorite “Asian Salmon” recipe may be; Seafood, Asian, Salmon, Healthy. To enter a tag, select the “Add” button next to the tag list. You will be asked to select either an “Existing Tag” or a new tag. This is to aid you in using existing tags as much as possible.

·        Meal Type – Allows you to tag a recipe as a Desert, Soup, Breakfast, Lunch, Dinner, etc. Again this is to aid you in searching for recipes in the future.

·        Main Ingredient – Allows you to choose one of the existing ingredients as the main one of the dish. Again this aid you in finding “All recipes you have for Salmon”, etc. NOTE: Only existing ingredients are allowed to be selected. So you will have to add ingredients before making a main ingredient selection.

·        Ingredients – Allows you to specify the amount and type of ingredients used in the dish. The amount field is free form so you can enter things like “2 cups” or “A pinch”. Once added you can edit existing ingredients without deleting them.

·        Instructions – Allows you to specify recipe instructions in a free flow step-by-step form. Once added you can edit existing instructions without having to delete them first. You are also allowed to delete individual instructions.

 

               Sorting, Filtering & Searching

 

iEat prides itself with being a user friendly searchable application, therefore iEat makes finding your favorite recipes a cinch.

 

Searching – You can use the search box at the top of iEat to search for recipes that contain any entered keywords you desire. Recipes that contain your entered keywords anywhere in the recipe information are displayed in the main window.

 

Filtering – Very similar to searching you can choose to easily filter recipes by certain recipe information in order to enable you to quickly find for example “All your soup recipes”, etc.

 

Sorting – Sorting allows you to sort the recipes currently displayed in the iEat main window. NOTE: You can also click the column header in the main view in order to sort the recipes.

 

Reset – At any time you can reset the iEat recipe view back to display all your recipes. Simply click the “Reset” button at the bottom of the Sort/Filter Pane.

                

               Printing & Backups

 

iEat wants you to be able to share and save your recipes as easily as you created them.

 

Backups – In order to backup all your recipes simply choose the backup button in the Main Toolbar. A dialog box will be displayed that will let you choose a save location. Recipes will then be saved in text (human readable) format for you to save elsewhere as you so desire.

 

Printing – iEat allows you to print one or multiple your recipes simply by selecting the desired recipes and choosing the “Print” button in the Main Toolbar.
